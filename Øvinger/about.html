Questions

  1. What does the server in the client server architecture do?
    the server is a software in a computer that respons to requests for the clients. The web page and database exists on the server´s locale system. the server retrieves and transmitts file to the client.
  2. What does the client do?
    the client sends different requests to a server through a agreed language or protocol, for example HyperText Transfer Protocol(HTTP). The server then respons to the request if the request is validated.
  3. What is an IP address?
    An internet protocol adress(IP address) is a numerical label to each device connnected to a computer network. An IP adress can be compared to a home adress. The IP adress consists of two parts, namely the network ID(street adress) and host ID(house number).
  4. What is a DNS?
    A domain name andress(DNS) identifies the intitution that owns the computer that contains the server. If you write ntnu.no, then the DNS machine of the organisation that owns the domain ".no" will search for the correct IP adress. Then at last your computer shows the homepage of ntnu.
  5. What is the difference between the Internet and WWW?
    The internet is a global network of networks while the World Wide Web(WWW) is one of many applications that work over the internet. WWW is a collection of documents and other resources which is linked by hyperlinks or Uniform Resource Identifier(URI). Web servers are accessed using HTTP or HTTPS protocols.
  6. What is HTML?
    HTML is a language that defines the structure of your content. This language consists of diffenret tags that can do different things, and often consists of opening and closing tags. HTML is often used combined with Cascading Style Sheets(CSS) and JavaScript(JS) when developing web pages.
